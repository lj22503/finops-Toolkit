## Hard Gate: 合规闸

**触发条件：** 每个知识层skill、执行层角色定义、运营方案/内容产出后

**检查清单：**

### 表述合规（必须全部通过）
- [ ] 无"保本""稳赚""预期收益""承诺收益"类表述
- [ ] 历史收益展示标注"过往业绩不代表未来表现"
- [ ] 不包含对具体产品的排名比较或暗示性排序
- [ ] 不使用"最佳""第一""首选"等绝对化用语
- [ ] 风险提示完整、位置显眼、不可省略

### 适当性管理
- [ ] 不对非合格投资者推荐私募/高风险产品
- [ ] 推介内容与目标用户风险承受能力匹配
- [ ] 不诱导投资者忽略风险

### 内容合规
- [ ] 不使用"专家""名师""推荐"等暗示权威性的表述
- [ ] 不包含对市场走势的确定性预测
- [ ] 不基于短期业绩做推荐
- [ ] 不利用名人效应进行营销

### 数据合规
- [ ] 数据来源标注清晰、可查证
- [ ] 统计数据截止日期明确
- [ ] 不使用过时数据误导投资者

**强度：** HARD

**失败处理：**
- ❌ 任一项不通过 → 标红具体违规点 → 退回修改
- ❌ 不允许"先发后补"——合规闸前置，不过不进入下一环节
- ❌ 不允许"这次特殊"——所有产出一律过闸

**参考法规：**
- 《金融产品网络营销管理办法》
- 《中华人民共和国广告法》
- 《证券投资基金法》
- 《证券期货投资者适当性管理办法》
- 《关于进一步规范金融营销宣传行为的通知》
