# /community-monitor — 社区舆情监控

监控基金社区讨论情绪、热点话题和异常信号，输出舆情简报。

## 调用

```
/community-monitor 天天基金 今天
/community-monitor 蚂蚁财富 本周
/community-monitor 全平台 舆情预警
```

## 工作流

### Step 1：数据采集

调用 **finops-community-monitor** skill，确定监控范围和平台。

- 明确目标平台（天天基金/蚂蚁财富/理财通/雪球）
- 明确时间段（今日/本周/近7天）
- 过**数据闸**：标注数据来源和采集时间

### Step 2：情绪分析

- 按正面/负面/中性分类讨论内容
- 标记情绪变化趋势（对比上一周期）
- 识别极端情绪爆发点

### Step 3：热点提取

- 提取用户集中讨论的主题和关键词
- 关联热点事件（市场波动/政策变化/产品表现）
- 识别KOL观点风向

### Step 4：输出舆情简报

```
## 舆情日报：[平台] - [日期]

### 情绪总览
正面 X% | 中性 Y% | 负面 Z%（较昨日 +-N%）

### 热点话题
1. [话题]（讨论量N条）
2. [话题]

### 预警信号
⚠️ [异常信号]

### 应对建议
- [建议]
```

### Step 5：过纪律层

- **合规闸**：简报中不得包含预测性言论
- **数据闸**：所有情绪数据标注统计口径
- **验证闸**：简报可被运营人员直接使用
