# /write-copy — 运营文案撰写

根据场景撰写合规的公募运营文案，支持短/中/长三类。

## 调用

```
/write-copy 短文案 短债基金 稳健型投资者
/write-copy 中文案 定投功能推广 存量持仓用户
/write-copy 长文案 投教：什么是债券基金 理财新手
```

## 工作流

### Step 1：确定文案类型

- **短文案**：Push/Banner（10-30字，突出利益点）
- **中文案**：详情页/落地页（100-300字，框架完整）
- **长文案**：投教/产品手册（1000-5000字，深度内容）

### Step 2：选择框架

**AIDA模型**（适合转化导向的文案）：
- Attention：场景/痛点切入
- Interest：产品方案/数据支撑
- Desire：信任信号/收益预期
- Action：行动引导

**SCQA模型**（适合投教/认知类文案）：
- Situation：当前市场/用户状态
- Complication：遇到的问题/矛盾
- Question：需要解决的核心问题
- Answer：提供的解决方案

### Step 3：输出文案

过**合规闸**：
- 不得出现"保本""稳赚""预期收益"
- 收益数据标注"过往业绩不代表未来表现"
- 风险提示完整不可省略

### Step 4：多平台适配

同一文案根据平台调整：
- 公众号：完整长文
- 小红书：拆成知识卡片
- 雪球：精炼为策略分析
- 企微：日签/短消息
