# /campaign-plan — 活动运营策划

根据增长目标策划完整的运营活动方案，覆盖5W1H。

## 调用

```
/campaign-plan 提升短债基金到期续投率 持有期30天用户 短债基金
/campaign-plan 新客首购转化 蚂蚁财富新注册用户 货币基金→短债
/campaign-plan 定投挑战活动 现有持仓用户 指数基金
```

## 工作流

### Step 1：明确活动目标

读取 `skills/knowledge/campaign-5w1h.md` 中的5W1H框架（Why/What/Who/When/Where/How）：

- **Why**：活动目的（拉新/促活/留存/转付费）
- **Who**：目标人群（新客/休眠户/高净值）
- **What**：活动规则（红包/抽奖/拼团/积分）
- **When**：活动周期与节奏
- **Where**：投放渠道（APP/社群/外部媒体）
- **How**：物料准备+技术开发+客服培训

### Step 2：方案设计

- 设计活动机制（用户做什么→得什么）
- 过**合规闸**：不得承诺收益、不得诱导短期交易
- 设计A/B测试对照组

### Step 3：输出活动方案

```
## 活动方案：[活动名称]

### 基本信息
目标 | 人群 | 规则 | 周期 | 渠道

### 用户路径
[用户从接触到参与的完整路径]

### 物料清单
- 上线前：文案/设计/开发/测试
- 上线中：监控/客服/调整
- 上线后：复盘/归档

### 成功标准
- 核心指标：[KPI]
- A/B测试：[方案A vs 方案B]

### 风险预案
- [风险场景] → [应对措施]
```

### Step 4：过纪律层

- **合规闸**：规则表述合规
- **数据闸**：KPI有基线数据支撑
- **验证闸**：方案可执行
