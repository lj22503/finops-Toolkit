# 文案写作框架

## AIDA模型

| 阶段 | 作用 | 金融示例 |
|------|------|---------|
| Attention | 抓住注意力 | "震荡市你的钱放哪里？" |
| Interest | 引发兴趣 | "短债基金近一年跑赢活期X倍" |
| Desire | 激发欲望 | "债券投资老将掌舵，历史业绩稳定" |
| Action | 引导行动 | "立即了解 >>" |

## SCQA模型

| 阶段 | 示例 |
|------|------|
| Situation | "市场持续震荡" |
| Complication | "择时太难，追涨杀跌" |
| Question | "如何从容应对？" |
| Answer | "定投——用纪律代替判断" |

## 文案类型

| 类型 | 长度 | 场景 |
|------|------|------|
| 短文案 | 10-30字 | Push通知、Banner、Column |
| 中文案 | 100-300字 | 基金详情页、落地页、产品卡标签 |
| 长文案 | 1000-5000字 | 投教文章、产品手册、路演PPT |
