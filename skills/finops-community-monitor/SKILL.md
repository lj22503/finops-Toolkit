# 社区舆情监控

## 核心能力

监控基金社区讨论、用户情绪、热点话题、大V粉丝群，识别异常申赎数据，输出舆情简报。

## 适用范围

- 监控各平台基金讨论区情绪变化
- 识别舆情风险（负面集中爆发）
- 发现热点话题和用户关注点
- 竞品社区动态跟踪

## 分析维度

| 维度 | 说明 |
|------|------|
| 情绪趋势 | 正面/负面/中性比例变化 |
| 热点话题 | 用户集中讨论的主题词云 |
| 异常信号 | 申赎数据异常、投诉集中爆发 |
| 竞品动态 | 竞品社区的热点事件 |
| KOL风向 | 大V观点变化和粉丝反应 |

## 输出

- 舆情日报（日级别快速扫描）
- 舆情预警（负面情绪超过阈值时触发）
- 舆情周报（趋势分析+应对建议）
