# 活动运营 5W1H 执行案

## Overview

每个活动必须有完整的 5W1H 规划，否则不能执行。

## When to Use

- 设计任何运营活动时
- 评审活动方案时
- 活动复盘时

## 5W1H Framework

### Why（活动目的）

- 本次活动的核心目标是什么？
- 是拉新、促活、转化还是品牌？
- 目标必须可量化

### Who（目标人群）

- 活动面向谁？
- 新用户还是老用户？
- 有什么特征/偏好？

### What（活动规则）

- 具体做什么？
- 参与门槛？
- 奖励机制？
- 周期和节奏？

### When（活动时间）

- 开始/结束时间
- 关键节点
- 运营排期

### Where（投放渠道）

- 在哪些渠道推广？
- 渠道特点是什么？
- 跳转路径？

### How（执行细则）

- 物料准备清单
- 技术开发需求
- 客服培训
- 风险预案

## Gate Rule

**活动规则和物料必须过合规闸才能上线。**

## Compliance Points

| 禁止 | 正确做法 |
|------|---------|
| "收益率最高" | "历史收益排名" + 风险提示 |
| "限时限量"制造FOMO | 真实限量或改为其他激励 |
| 承诺奖励 | 说明奖励条件 |

## Anti-Patterns

- 活动规则模糊 → 用户投诉
- 没有风险预案 → 出现问题手忙脚乱
- 活动结束没复盘 → 经验不沉淀
