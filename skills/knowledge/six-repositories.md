# 六库资产沉淀法

## Overview

建立并持续更新六大仓库：素材、数据、资料、模板、品控、工具。

## Six Repositories

| 仓库 | 内容 | 用途 |
|------|------|------|
| 素材库 | 图片/Banner/文案素材 | 快速复用 |
| 数据库 | 运营数据/用户数据 | 分析决策 |
| 资料库 | 行业报告/竞品资料/培训材料 | 知识沉淀 |
| 模板库 | 活动模板/文案模板/分析模板 | 标准输出 |
| 品控库 | 合规检查清单/质量标准 | 底线保证 |
| 工具库 | 软件/脚本/自动化流程 | 效率提升 |

## When to Use

- 新人入职时 → 快速上手
- 大型活动筹备时 → 复用已有素材
- 出品不稳定时 → 查品控库

## Expected Outcomes

| 指标 | 预期效果 |
|------|---------|
| 新人上手时间 | 从 1 周缩短到 1 天 |
| 活动筹备时间 | 缩短 50% |
| 出品质量底线 | 有品控库兜底 |

## Anti-Patterns

- 只建不用 → 仓库变成死库
- 建了不更新 → 内容过时，比没建更危险
