# 金融营销合规审查

## Overview

金融产品营销必须符合监管要求，所有内容/活动发布前必须经过合规审查。

## When to Use

- 审查营销内容合规性时
- 生成合规营销方案时
- 判断表述是否违规时

## Review Sequence（必须按顺序执行）

```
1. 资质检查（最先）→ 是否有相应业务资质？无 → 直接违规
2. 禁止行为检查 → 触碰8项禁止行为？是 → 直接违规
3. 内容真实性检查 → 数据/表述真实准确？否 → 标注违规
4. 关键信息完整性 → 风险提示/费用/期限完整？缺 → 标注需修改
5. 适当性管理 → 匹配目标人群风险等级？不匹配 → 标注需修改
6. 渠道合规性（最后）→ 跳转链接/合作平台合规？
```

**原则：先查致命项（资质/禁止行为），再查一般项。**

## 核心法规

- 《金融产品网络营销管理办法》（2026年9月30日实施）
- 《中华人民共和国广告法》
- 《证券投资基金法》
- 《证券期货投资者适当性管理办法》

## Key Rules

### 🚫 禁止行为（8项）

1. 明示或暗示保本、承诺收益
2. 使用诱导性用语（限时限量、抢购）
3. 使用绝对化用语（最佳、第一、首选）
4. 包含对市场走势的确定性预测
5. 不基于短期业绩做推荐
6. 利用名人效应营销
7. 虚假或误导性内容
8. 其他违反监管规定的行为

### 资质要求

- 开展营销业务需相应资质
- 无资质 → 直接判定违规

### 适当性管理

- 不对非合格投资者推荐高风险产品
- 推介内容与目标用户风险承受能力匹配
- 不诱导投资者忽略风险

### 数据合规

- 数据来源标注清晰、可查证
- 统计数据截止日期明确
- 不使用过时数据

## Compliance Markers

| 表述 | 合规性 |
|------|-------|
| "过往业绩不代表未来表现" | ✅ 需标注 |
| "投资有风险" | ✅ 需完整 |
| "保本" | ❌ 违规 |
| "预期年化收益5%" | ⚠️ 需上下文 |
| "限时抢购" | ❌ 可能违规 |

## Gate Rule

**所有运营产出必须过合规闸才可发布。**
