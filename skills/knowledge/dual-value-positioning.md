# 双向价值定位法

## Overview

分析竞争优势的方法：对外找市场空白，对内盘点自身优势。

## When to Use

- 竞品分析时
- 确定产品/专题切入点时
- 避免同质化竞争时

## Core Framework

### 对外分析（市场/竞品）

| 维度 | 分析内容 |
|------|---------|
| 竞品在做什么 | 产品/服务/营销策略 |
| 用户未被满足的需求 | 痛点/缺口 |
| 市场空白 | 未被覆盖的人群/场景 |

### 对内分析（自身）

| 维度 | 分析内容 |
|------|---------|
| 投研能力 | 专业度/研究深度 |
| 客户基数 | 用户规模/质量 |
| 资源禀赋 | 渠道/品牌/数据 |

### 定位决策

找到「我做得比竞品好 + 用户真正需要」的交叉点。

## Application

**示例：**
- 基金详情页优化 → 分析竞品页面的信息结构，找到我们能补充的维度和更清晰的呈现方式
- 投教内容专题 → 找竞品没覆盖的投资理念或角度

## Anti-Patterns

- 只看竞品，不看自身优势 → 容易模仿但没差异化
- 只看自身，不看市场 → 容易自嗨但用户不需要
