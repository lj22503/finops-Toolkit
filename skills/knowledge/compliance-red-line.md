# 合规红绿灯意识

## Overview

金融营销内容必须遵守合规红线，任何产出发布前必须过合规审查。

## When to Use

- 创建任何内容（文案/图片/视频）
- 设计任何活动方案
- 任何用户触达（Push/短信/弹窗）

## Compliance Checklist

### 🚫 禁止项（直接违规）

| 禁止表述 | 示例 |
|---------|------|
| 保本/保息 | "本金保障，稳赚不赔" |
| 承诺收益 | "预期年化收益 5%"（基金不可） |
| 诱导性用语 | "限时限量"、"抢购"制造不合理 FOMO |
| 绝对化用语 | "最佳"、"第一"、"首选" |
| 暗示性排名 | "收益率最高的前三名基金" |
| 个股推荐 | "买茅台" |

### ⚠️ 需谨慎

| 表述 | 要求 |
|------|------|
| 历史收益展示 | 必须标注"过往业绩不代表未来表现" |
| 风险提示 | 必须完整、位置显眼、不可省略 |
| 数据来源 | 必须标注来源和截止日期 |

### ✅ 合规表述

- "年化收益的 historical average is X%"
- "基金过往业绩不预示未来表现"
- "投资有风险，决策需谨慎"

## Gate Rule

**所有运营产出必须过合规闸才可发布。**

## Anti-Patterns

- "别人都这么写，我们也可以" → 不是免责理由
- "先发后补" → 合规是一票否决的前置条件
- "审核通过了没问题" → 每次产出都要审，不能套模板
